Parses PostgreSQL tuples, with support for:

* composite tuples
* arrays of tuples

## Installing

```
$ npm install pg-tuple
```

## Testing

First, clone the repository and install DEV dependencies.

```
$ npm test
```

Testing with coverage:
```
$ npm run coverage
```
